# Pet Adoption Platform

## Project Overview
The **Pet Adoption Platform** is a CRUD-based application designed to streamline the pet adoption process. It provides a seamless experience for both customers and staff.

### Key Features:
- **Customer Site**: Users can browse available pets for adoption, view pet details, and easily submit adoption requests.
- **Staff Portal**: Secure login for staff to manage pet listings, including Create, Read, Update, and Delete (CRUD) operations. Staff can also manage adoption requests and update pet information.
- **Database Management**: A well-structured MySQL database supports the efficient storage and management of pet data, customer information, and adoption records.

This project was developed as part of the **Database Application Development** course. The team consists of three members, and the project significantly helped enhance skills in **database design**, **authentication**, and **CRUD operations**.

## Features
- **Browse Available Pets**: Customers can view and search for pets that are up for adoption.
- **User-friendly Interface**: A clean and intuitive design enhances the user experience, making it easy to navigate through pet listings.
- **Staff Portal**: Provides a secure authentication system for staff members to manage pet listings and adoption requests.
- **Full CRUD Management**: Staff can create, read, update, and delete pet listings, ensuring the data is always up to date.
  
## Skills & Technologies:
- **Database**: MySQL for efficient data storage and management.
- **Backend**: Java for implementing the backend logic and managing CRUD operations.
- **Web Development**: Integration of frontend and backend technologies to ensure a responsive and seamless user experience.
